# 指针式仪表读数识别系统

## 1. 项目简介

本项目针对工业场景中的指针式仪表自动读数问题，
利用计算机视觉技术实现仪表图像中的表盘定位、
指针检测以及数值转换。

项目目标是实现输入一张仪表图片，
自动输出对应的仪表读数。

---

## 2. 项目背景

传统工业现场仪表巡检通常依赖人工读取，
存在效率低、实时性不足以及人工误差等问题。

通过计算机视觉技术，可以实现仪表状态自动感知，
提高工业巡检智能化水平。

---

## 3. 技术路线

整体流程：
仪表图像输入 -> 图像预处理 -> 表盘区域检测 -> 指针定位 -> 角度计算 -> 刻度映射 -> 输出读数


---

## 4. 技术栈

- Python
- OpenCV
- NumPy
- Matplotlib

后续根据实验需求进行算法优化。
